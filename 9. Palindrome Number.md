## Convert to string then use two pointers:

```java
class Solution {
    public boolean isPalindrome(int x) {
        String s = Integer.toString(x);
        int len = s.length();
        if(len<2) return true;
        int l=0,r=len-1;
        while(l<r){
            if(s.charAt(l)!=s.charAt(r)){
                return false;
            }
            l++;
            r--;
        }
        return true;
    }
}
```

## Reverse half

```java
class Solution {
    public boolean isPalindrome(int x) {
        if(x<0||(x%10==0 && x!=0)) return false;
        int rev=0;
        while(x>rev){
            rev=rev*10+x%10;
            x/=10;
        }
        return x==rev|x==rev/10;
    }
}
```
