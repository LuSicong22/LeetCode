## Method 1: Greedy
```java
class Solution {
    public int maxProfit(int[] prices) {
        int len=prices.length;
        if(len<2){
            return 0;
        }
        
        int res=0;
        for(int i=1;i<len;i++){
                res+=Math.max(prices[i]-prices[i-1],0);
        }
        return res;
    }
}
```
