## Method 1: Binary Search
Go
```go
func searchInsert(nums []int, target int) int {
    left, right :=0, len(nums)-1
    if target > nums[right] {
        return right+1
    }
    for left < right {
        mid := left + (right - left)>>1
        if nums[mid] < target {
            left = mid+1
        } else {
            right = mid
        }
    }
    return left
}
```
